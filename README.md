# Public Policy RAG Chatbot

Starter workspace for a public-policy chatbot using React, Express, and FastAPI.

## Projects
- `frontend`: React client
- `backend`: Express API gateway
- `ai-service`: FastAPI RAG service
- `docs`: architecture notes

## Suggested next steps
1. Install React, Express, and FastAPI dependencies.
2. Start the FastAPI service on port `8000`.
3. Start the Express backend on port `5000`.
4. Start the frontend with Vite.
5. Add PostgreSQL and vector database configuration.

## Run commands
- Frontend: `npm run dev`
- Backend: `npm run dev`
- AI service: `py -m uvicorn app.main:app --reload`
